# Security Policy

## Supported Versions

Currently, the latest version of Animation Galaxy is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the repository owner via their GitHub profile contact information. All security vulnerabilities will be promptly addressed.

### Best Practices implemented in this project:
- No external dependencies (Pure CSS/JS) to minimize supply chain attacks.
- Strict Content Security Policy (CSP) compatibility (inline styles are used for dynamic previews, but core logic is separate).
- No user data collection.
